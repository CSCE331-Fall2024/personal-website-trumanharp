BG3 Warforged

This is a mod for the game Baldur’s Gate 3 (which won game of the year). It adds a new playable race to the game, the warforged. In this project I learned to analyze json files and learned a lot about large game structure and processes. This project in total has taken probably close to 50 hours to complete, and has been downloaded and used by thousands of people, so I'm very proud of it. I'm still updating it relatively regularly, and am looking for someone to help me create models. I have created a couple more mods for Baldur's Gate 3 which are private or unfinished, as well as contributed to a couple community mods that are XML and Noesis GUI files.

Wordle Project

This is a wordle clone that I made in java for my CSCE 314 class last year. I learned about java and user interface, and a lot about graphical user interface design. This project took me several hours to complete, and was part of my final for the class. The game is very user friendly, it has a clean GUI and helpful popup warnings and notifications. I followed object oriented programming, and used collections, inhertiance, a HashMap, and a custom data structure. It also has a File IO system for game saves for replayability.

Project Zomboid Mod

This is a mod I wrote for the indie game Project Zomboid. It adjusts the tailoring perk in the game and adds some crafting recipes, like it’s namesake, crafting pillows. I learned a lot about small game structure and design, as well as Lua scripting and how to learn a proprietary info structure. I also began to learn a bit of 3d modeling and rendering, which is something I had never done before. The documentation for modding Project Zomboid is pretty rudimentary, so I had to learn a lot by dissasembling existing mods and the actual game files.
