BG3 Warforged

This is a mod for the game Baldur’s Gate 3 (which won game of the year). It adds a new playable race to the game, the warforged. In this project I learned to analyze json files and learned a lot about large game structure and processes. I have a couple similar, unfinished mods that are still private.

Wordle Project

This is a wordle clone that I made in java for my CSCE 314 class last year. I learned about java and user interface, and a lot about graphical user interface design.

Project Zomboid Mod

This is a mod I wrote for the indie game Project Zomboid. It adjusts the tailoring perk in the game and adds some crafting recipes, like it’s namesake, crafting pillows. I learned a lot about small game structure and design, as well as Lua scripting and how to learn a proprietary info structure.
